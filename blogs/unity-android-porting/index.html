unity need sdk and jdk.

for the sdk :

u can download sdk by downloading the android studio and waiting for it to install everything, then copying the files at "C:\Users\username\AppData\Local\Android\sdk\" to some folder, and u should have these folders (.temp, build-tools, licenses, patcher, platforms, platform-tools, tools, .knownPackages(file) ). after uninstall android studio and download this command line tools (go to the very down of the page) to be able to download any version u want of the android builds (so if u have the 11th build version that is API level 30 u'll be able to port an android app with unity and use it in android 11) and here is a guide on how to use the sdk manager with the command line tools but what u'll need essentially is to go to the command line tools folder frm zip u downloaded and go to "bin" folder and (in windows) to open a command in that folder go to the folder path and type "cmd" press enter, then type in the command window this : sdkmanager "platforms;android-28" --sdk_root=C:\Users\username\Downloads\programz\sdks . u can change the "sdk_root" to where u putted the copied files frm the android studio and for the index "28" it represent the API level and u can change it based on this guide. and after that u can locate from unity the sdk by pasting the sdk folder path to "Preferences > External Tools" (under android).

and for the jdk :

u will jst have to download the open jdk extract and save to some folder and locate too in unity.
